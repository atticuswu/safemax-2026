## 一、 核心底層邏輯：4.7% SAFEMAX

根據 Bill Bengen 2025 年的研究，**4.7%** 是通過百年歷史壓力測試（含大蕭條、惡性通膨）的「全域安全最大提領率」。

- **基準點：** 4.7% 是歷史上的「地板」，對應最差的退休時點（如 1968 年）。
- **兩大變數：** 退休開場時的 **席勒 CAPE 值** 與 **前五年通膨率**，決定了你的個人 SAFEMAX 能否向平均值 7.1% 靠攏。

---

## 二、 動態提領框架：每年根據市場水位重算支出

提領不是固定金額 + CPI 調整，而是**每年根據當年組合市值 × CAPE 對應 SWR 重新計算**。市場漲了自動花更多，市場跌了自動縮減，從根本切斷「熊市持續失血」的惡性循環。

### 1. 每年支出計算公式

$$\text{當年支出} = \text{當年組合市值} \times \text{CAPE 對應 SWR}$$

### 2. CAPE 對應提領率（SWR）

| 市場估值 (CAPE) | 適用 SWR | 戰略邏輯 |
| :--- | :--- | :--- |
| **> 30（昂貴）** | **4.7%** | 防守模式，應對潛在的報酬順序風險。 |
| **15 – 30（合理）** | **5.2%** | 標準模式，資產增長機率較高。 |
| **≤ 15（便宜）** | **6.0%** | 進攻模式，此時提領對資產長線損害最小。 |

> **與傳統 Bengen 的差異：** 傳統做法是「退休第一年定好金額，之後逐年乘以 CPI」，本策略改為每年動態重算，支出與資產規模連動，避免固定高提領在熊市不斷侵蝕本金。

## 三、 核心洞察：質押不等於更高終局資產，但在 CAPE ≤ 15 時價值最大

回測發現，**質押策略並不能顯著提高去世時的終局資產**，原因是：

- 賣股模式：組合市值縮小，但無負債
- 質押模式：組合市值維持，但負債滾利息

兩者的「淨資產 = 市值 − 負債」走勢趨於收斂，加上動態 SWR 讓支出與市值連動，終局差距很小。

**質押的真正價值在三件事：**
1. CAPE ≤ 15 時，股票最便宜、未來期望報酬最高，此時被迫賣股代價最大
2. 台灣稅制下借款不計所得，節省資本利得稅與二代健保補充保費（實質現金優勢）
3. 熊市中「股數從未減少」的心理錨定，防止恐慌性操作

### 精煉後的執行邏輯：CAPE 條件式質押

| CAPE 水位 | 預設執行模式 | 邏輯 |
| :--- | :--- | :--- |
| **> 15（正常至昂貴）** | **動態賣股** | 賣出溢價或合理價股票，不付利息，簡單乾淨 |
| **≤ 15（歷史便宜）** | **啟動質押** | 股票最珍貴，每股未來期望報酬最高，借錢生活而非賣股 |

> **為何這才是真正的反脆弱？** 質押是昂貴工具（有利息成本），只在「不賣股的價值最高」的時候使用，才能讓工具成本與效益精確匹配。CAPE ≤ 15 的時點（如 2009 年底部）正是護欄最可能觸發的時點，兩個機制自然重合，不需額外判斷。

### 質押模式雙重護欄

啟動質押後，仍受以下兩個條件監控，任一觸發即切回賣股：

- **條件 A（維持率護欄）：** `組合市值 / 累積借款 × 100% < 300%`
- **條件 B（債務天花板）：** `累積借款 ≥ 初始退休資產 × 25%`

> $$\text{維持率} = \frac{\text{總資產市值}}{\text{累積借款金額}} \times 100\% \ge 300\%$$
>
> $$\text{借款上限} = \text{初始退休資產} \times 25\%$$

## 四、 風險管理與護欄 (Safety Guardrails)

### 1. 護欄觸發：雙重條件任一成立即切換

| 觸發條件 | 程式代號 | 說明 |
| :--- | :--- | :--- |
| 維持率 < 300% | `"maintenance"` | 資產相對負債不足，市場下跌造成 |
| 累積借款 ≥ 初始資產 × 25% | `"debt-cap"` | 借款達硬上限，主動截斷滾雪球 |

### 2. 護欄模式執行公式（賣股補缺口）

停止新增借款，改用下列公式決定每年賣股金額：

**Step 1：計算當年淨股息**

$$\text{淨股息} = \text{當年股息收入} - \text{累積借款利息}$$

**Step 2：決定賣股金額**

| 情境 | 條件 | 賣股金額 |
| :--- | :--- | :--- |
| 股息仍能覆蓋利息 | 淨股息 > 0 | `max(0, 當年支出 − 淨股息)` |
| 股息連利息都不夠 | 淨股息 < 0 | `當年支出 + \|淨股息\|`（生活費 + 利息缺口同步補足） |

> **白話解釋：** 先讓股息去抵扣借款利息，剩下不足的部分才賣股補齊。此模式下累積借款金額**不再增加**，靠股息持續自然攤銷利息。

**Step 3：持續執行，直到維持率回升至 300% 以上且借款低於天花板**，方可恢復質押模式。

### 3. 去槓桿機制（牛市主動還債）

在牛市後期（CAPE > 30），應評估將部分「溢出」的資產報酬用於還清質押本金，將維持率推回 500% 以上，同時讓累積借款遠離天花板，為下一場熊市保留完整的質押空間。

## 五、 壓力測試：1999 年退休情境

最壞的歷史起點（CAPE 41.3）：2000–2002 網路泡沫（-43%）+ 2008 金融海嘯（-37%），連遇兩次重擊。

- **60 歲（2009 年）是壓力頂點：** 維持率跌破 300%（或觸及債務天花板），護欄自動啟動，切換賣股模式。
- **此時 CAPE 僅 15.1，市場處於歷史低估區**，賣股雖痛，每年賣出的僅是「生活費 − 淨股息」的缺口，非整批清倉。
- **後續強力反彈：** 動態 SWR 在低點自動壓縮支出，保護存活本金，2010 年後市場複利修復資產。

**關鍵應對原則：** 不做系統外的決策（不恐慌全清）、考慮暫時降支出至 4.7% 水準、等待維持率自然回升後恢復質押。

## 六、 2026 年實務備忘錄

> **執行時機：** 本策略預計於 2029 年 Grand Park 落成後，辛亥路與哈爾濱街陸續出脫、流動金融資產比重大幅提升後，才進入全速執行階段。2029 年前以模擬回測為主，逐步建立可質押部位。

1. **契約管理：** 注意台灣券商質押通常為「1.5 年 + 1.5 年」滾動。須確保有「備援券商」應對合約到期時的轉質需求。
2. **標的監控：** 質押標的必須是大盤型 ETF。若標的被剔除於券商質押清單，需有備用的現金緩衝空間。
3. **債務天花板設定：** 預設 25%，可依個人風險承受度在回測程式中調整。建議從壓力最大的歷史起點（1999 年）反覆測試，找出不觸發斷頭的最大值。
4. **心態修正：** 記住，4.7% 是保險，質押是工具。真實的財務自由不在於資產數字，而在於「面對崩盤時依然能穩定產生現金流」的掌控感。

## 七、 批判性觀點與反駁

> 這套策略最完美的平衡點在於「股息覆蓋利息」。一旦全球進入超高利率時代（如質押利率 > 5%），這套邏輯將面臨挑戰，屆時需果斷回歸傳統的 Bengen 提領法。

### 主要風險清單

| 風險因素 | 影響程度 | 應對措施 |
| :--- | :--- | :--- |
| 質押利率 > 股息殖利率 | 高 | 回歸賣股提領，停止質押 |
| 市場連續下跌 > 30% | 高 | 維持率護欄 + 債務天花板雙重保護 |
| 退休時 CAPE 極高（> 40） | 高 | 天花板設保守值（如 15–20%），降低最大槓桿 |
| ETF 被剔除質押清單 | 中 | 備援券商 + 現金緩衝 |
| 合約到期無法續約 | 中 | 提前 6 個月安排轉質 |
| 惡性通膨 CPI > 5% | 中 | 動態 SWR 機制自動壓縮支出，部分緩衝 |

## 八、 回測程式功能說明（2026-04-01 版）

### 可回測資產

| 資產 | 說明 | 代理期間 |
| :--- | :--- | :--- |
| **SPY** | S&P 500，含 Shiller CAPE | 全歷史（1966–2023）真實數據 |

| **0050** | 台灣加權 / 0050 | 2003 前採台灣加權指數代理 |

> ⚠️ **重要限制：** SWR（提領率）計算永遠使用**美股席勒 CAPE**作為基準，即使選擇 VT 或 0050 回測亦然。台灣市場估值結構與美股不同，0050 模擬結果應視為參考，而非精確預測。

### CAPE 條件式質押 UI

程式提供三段切換：

| 設定 | 效果 |
| :--- | :--- |
| **不限** | 無論 CAPE 高低均可質押（傳統模式） |
| **CAPE <** + 數值 | 僅市場便宜時才質押（推薦：CAPE < 15） |
| **CAPE >** + 數值 | 僅市場昂貴時才質押（不建議） |

推薦設定：**CAPE < 15**，對應「股票最珍貴、不賣股價值最高」的歷史低估水位。

### Tooltip 護欄原因

賣股模式觸發時，Tooltip 會顯示觸發原因：
- `維持率`：市值/借款比率跌破門檻
- `借款上限`：累積借款達初始資產 × 上限比例
- `CAPE條件`：當前 CAPE 超出質押條件範圍

### 其他主要參數

- **質押年度 / 賣股年度**：列出每個年份的策略模式，可驗證 CAPE 條件是否正確觸發
- **終局三指標**：淨資產、資產市值、累積債務分別顯示
- **開始計算**按鈕：手動觸發重算，避免每次調整參數都自動跑模擬

### 0050 歷史數據的特殊性

台灣股市歷史波動極端（1986 +90%、1988 +119%、1990 -53%、2008 -46%、2009 +78%），遠超 SPY。以 0050 回測時應注意：實際退休後的組合波動可能比歷史數據溫和（0050 成立後才逐漸分散），但極端情境下的壓力測試仍有參考價值。
